# 🍏 iFruit Landing Page

A premium GTA-inspired iFruit landing page built with HTML and CSS.

## 🚀 Live Demo
https://iFruit-landing-page.netlify.app/

## ✨ Features

- Responsive Design (Desktop, Tablet & Mobile)
- Modern Dark UI
- Smooth Hover Effects
- Glassmorphism-inspired Components
- Pricing Cards
- FAQ Section
- Reviews Section
- Responsive Navigation
- Clean HTML & CSS Structure

## 🛠️ Built With

- HTML5
- CSS3
- Remix Icons
- Font Awesome
- Google Fonts (Inter)
